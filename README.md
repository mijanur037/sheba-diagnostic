# 🏥 Sheba Diagnostic & Consultation Center
# Live Site: https://mijanur037.github.io/sheba-diagnostic/

A modern, fully responsive, and interactive landing page designed for Sheba Diagnostic & Consultation Center, located in Matlab South, Chandpur. This project provides a seamless user experience for patients to view doctor schedules, explore health packages, read success stories, and book appointments directly via WhatsApp.

✨ Key Features
📱 Fully Responsive Design: Optimized for all devices (Mobile, Tablet, Desktop) using custom CSS Grid and Flexbox.

📅 Dynamic Appointment Booking: An interactive modal that captures patient details and redirects them to a formatted WhatsApp message for easy booking.

🩺 Comprehensive Medical Services: Dedicated sections for Departments, Specialist Doctors, and detailed Health Packages.

🏠 Home Sample Collection: Features an embedded vertical video layout to promote at-home diagnostic services.

💬 Patient Reviews & Success Stories: Includes a scrolling review carousel and a custom modal for users to submit their own feedback directly to the clinic's WhatsApp.

📖 Health Blog: A visually appealing grid layout highlighting medical articles and health tips.

🎨 Modern UI/UX: Utilizes glassmorphism effects, smooth @keyframes animations, floating action buttons, and a sticky bottom navigation bar for mobile users.

🛠️ Tech Stack
HTML5: Semantic and accessible page structure.

CSS3: Custom styling, CSS variables for theming, animations, and responsive media queries (No external CSS frameworks used).

Vanilla JavaScript: Handles mobile menu toggling, modal popups, dynamic review rendering, and WhatsApp URL generation.

🚀 How to Use
This project is completely front-end and requires no complex build tools or server setup.
